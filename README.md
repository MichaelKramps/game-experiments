# Game Prototypes

Quick game ideas to build and evaluate in under a week.

## Status Key
- `idea` — not started
- `in-progress` — currently building
- `promising` — worth pursuing further
- `abandoned` — not fun / not worth it

## Games

| Game | Status | Notes |
|------|--------|-------|
| _template | — | Phaser 3 starter, copy this for new games |

## Starting a new game
1. `cp -r _template my-game-name`
2. Open `my-game-name/index.html` in a browser
3. Edit `game.js` and refresh to iterate
